• Step 1: Triggering VOD Encoding with Bitmovin Encoder

    • To initiate the encoding process, we will trigger an AWS Lambda function in this demo, which will spawn encoding instances and upload the encoded output to an S3 bucket.

    Encoding Dashboard

Demo Architecture

  • Step 2: Monitoring Encoding Progress

    • Once the encoding process has started, you can monitor the progress on our dashboard (https://bitmovin.com/dashboard). Please wait until the encoding status is displayed as "FINISHED."

  • Step 3: Configuring Automated Workflow for SSAI Manifest

    • After the encoding process is complete, an automated workflow is configured to place the encoded output in AWS MediaTailor. This allows us to obtain an SSAI VOD manifest from AWS MediaTailor.

SSAI Playback with Bitmovin Player

It is now time to play the SSAI manifest, which can be done using Bitmovin Player. The left column of the web application will display the player screen while the right column will showcase the player log, which demonstrates how we extract events from the SSAI stream via metadata.
